jQuery Sliding Deck

Full screen stacking content — display your website content like a deck of cards

  • Uses native browser scrolling — does not hijack the scrollbar!
  • Not just full screen — can be used on small sections
  • Only about 6 KB — for all required JavaScript and CSS
  • Mobile friendly — stacks your content only when the screen is big enough

Forked from rgrandi/jquery.stacked-strips, created by Jon Pontet.

How to use

1. Load jQuery and include jQuery Sliding Deck plugin files

You will need to use jQuery (version 3.1.1 or greater) and the JavaScript and CSS file of jQuery Sliding Deck as follows...

In the <head> tag:


<!-- jQuery Sliding Deck -->
<link rel="stylesheet" href="jquery.sliding-deck.min.css" />

Before the </body> tag:


<!-- jQuery 3.1.1+ -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>

<!-- jQuery Sliding Deck -->
<script src="jquery.sliding-deck.min.js"></script>

2. Setup your HTML

You need at least 2 HTML elements the same (i.e. matching to the same jQuery selector) to make jQuery Sliding Deck work.
For the most semantic HTML, use <section> tags:


<section>
  <h2>Section 1</h2>
  <p>Section content</p>
</section>
<section>
  <h2>Section 2</h2>
  <p>Section content</p>
</section>

3. Call the plugin

You are now ready to initialise jQuery Sliding Deck with your preferred options:


$(document).ready(function() {
  $("section").slidingDeck();
});

Customising

1. Options

You can use any or all of these options when initialising jQuery Sliding Deck:

Variable Default Type Description
strip_size "full" string/int Fill the whole screen by default or specify a pixel value to restrict the height.
active_position 50 int At what height in percent the active class will be added to the current slide.
after_class false boolean Set to true to add the name class to the previous slide, when the active class is added to the current slide.
stackActivationWidth 781 int At what width in pixels elements will commence stacking. Before this width, the elements will scroll as normal. Use this to specify the minimum width your content needs to display in its entirety in the viewport.
showNav true boolean Set to false if you do not want the dot navigation.
navSelector "#top-nav" string Specify a custom jQuery selector for the navigation container.
showNextSlideButton true boolean Set to false if you do not want the next slide button.
showNextSlideButtonSelector "#next-slide" string Specify a custom jQuery selector for the next slide button container.

2. Defaults

All default options:


$(document).ready(function() {
  $("section").slidingDeck({
    strip_size: "full",
    active_position: 50,
    after_class: false,
    stackActivationWidth: 781,
    showNav: true,
    navSelector: "#top-nav",
    showNextSlideButton: true,
    showNextSlideButtonSelector: "#next-slide"
  });
});
